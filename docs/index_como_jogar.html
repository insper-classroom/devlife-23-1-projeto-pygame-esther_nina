Como Jogar

Para jogar, você deve utilizar o mouse para desenhar as plataformas na tela. Quanto mais longas e seguras forem as suas plataformas, melhor será sua pontuação e maiores serão suas chances de vencer o jogo. Tome cuidado para não deixar a bola cair, ou você terá que começar novamente.

Para baixar o jogo, basta clicar no link "DOWNLOAD" abaixo. O download virá em formato de pacote zip, que deve ser descompactado para que você possa jogar. O jogo é compatível com Windows e Mac, porém é necessário ter instalado o aplicativo "Visual Studio Code" e a linguagem de código "Python".
